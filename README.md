# Basic Auth Project

This is a simple authentication project built with:

- Node.js
- Express
- MongoDB
- Mongoose
- JWT
- bcrypt
- EJS

## Features

- User Registration
- User Login
- Password Hashing
- JWT Authentication
- Protected Route
- Logout

## How to Run

1. Install dependencies

npm install

2. Start server

nodemon app.js

Server runs on:
http://localhost:3000

This project is made for learning backend development.
